Git worktree review in one terminal UI.

tui-worktree is a git worktree TUI and worktree diff viewer for reviewing AI-generated code changes across linked Git worktrees.

View on GitHub Install

Review worktrees

Browse linked Git worktrees, changed files, and diffs without jumping between shell commands.

Inspect AI changes

Keep agent-generated code isolated, compare the result, and decide what to merge or delete.

Stay in flow

Open files in your editor, create PRs or MRs, merge branches, and delete disposable worktrees.

Install

brew install overthinker1127/tap/tui-worktree
